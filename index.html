eniLINK

Linked Data Platform

eniLINK is a web platform for linked data applications based on the KOMMA framework and Lift

Documentation Discussion Github

eniLINK is open-source software by Fraunhofer IWU.
The development has received funding by the European Union (European Regional Development Fund) and Germany's Free State of Saxony within the Cluster of Excellence "Energy-Efficient Product and Process Innovation in Production Engineering" (eniPROD®) and others.
